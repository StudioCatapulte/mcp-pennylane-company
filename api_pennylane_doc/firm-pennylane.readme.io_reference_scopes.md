Scopes are used to limit a token's access to Pennylane data.

**Required scopes per endpoint**

Here's our mapping of API endpoints and their required scopes:

| Scopes | Endpoints |
| --- | --- |
| companies:readonly | List companies<br>Show companies |
| fiscal\_years:readonly | List company's Fiscal Years |
| trial\_balance:readonly | Get the trial balance of a company |
| exports:agl | Create an analytical general ledger export<br>Retrieve an analytical general ledger export |
| exports:fec | Create a FEC export<br>Retrieve a FEC export |
| dms\_files:readonly | List DMS files<br>List DMS files changelog events |
